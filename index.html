Media Server

Description: This home media server is designed to be run on a Raspberry Pi (3b or newer versions) to stream media content such as videos, pictures and music over any DLNA compatible device over the network. It uses a torrent client to download content directly to an external HDD attached to the RPi, from where any user on the network can stream that content on a DLNA compatible device such as any smart TV.
